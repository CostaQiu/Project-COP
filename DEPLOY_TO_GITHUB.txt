STEP 5: Connect to GitHub and Push
================================================================================

================================================================================
AUTHENTICATION OPTIONS
================================================================================

Option A: Personal Access Token (Recommended for HTTPS)
1. Go to: https://github.com/settings/tokens
2. Click "Generate new token"
3. Select "repo" scope
4. Copy token and use as password when prompted

Option B: GitHub CLI
1. Install: https://cli.github.com/
2. Run: gh auth login
3. Follow prompts

Option C: SSH Key (Advanced)
1. Generate key: ssh-keygen -t ed25519 -C "your@email.com"
2. Add to GitHub: https://github.com/settings/keys
3. Use SSH URL: git@github.com:YOUR_USERNAME/Project-COP.git

================================================================================
TROUBLESHOOTING
================================================================================

Error: "fatal: not a git repository"
Solution: Run "git init" first

Error: "Please make sure you have the correct access rights"
Solution: Check your authentication (token/SSH key)

Error: "src refspec main does not match any"
Solution: Use "git branch -M main" to rename branch

Error: "The local repository is out of date"
Solution: 
git fetch origin
git merge origin/main

Success! But no files visible:
Solution: Wait 1-2 minutes for GitHub to process, then refresh
